# `view`

View a gitignore file

## Aliases

`show`, `get`

## Description

View a gitignore file in the terminal. Use the "list" command to view all available gitignore templates.

## Usage

```
gh-gitignore view
```



## Examples

```
  gh gitignore view Go
  gh gitignore view Go Python
```
