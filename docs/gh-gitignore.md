# `gh-gitignore`

Generate .gitignore files for your projects



## Description

Generate .gitignore files for your projects using GitHub's gitignore template API.

## Usage

```
gh-gitignore
```



## Examples

```
  gh gitignore list
  gh gitignore view Node
  gh gitignore create Rust
```
